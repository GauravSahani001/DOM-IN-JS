```BOM IN  JS
 üîç BOM Kya Hai ?
  BOM ka full form hai Browser Object Model.Iska use browser ke elements(window, history, screen, navigator, location, etc.)
ko access aur manipulate karne ke liye hota hai.

 üß± BOM ke Important Objects:
 1. window object ‚Äì Sabse bada object hota hai.
 alert("Hello"); // actually is window.alert("Hello");
 console.log(window.innerWidth); // Browser window ki width

 2.navigator object ‚Äì User ke browser ke baare mein info deta hai.
console.log(navigator.userAgent); // Konsa browser use ho raha hai

 3. screen object ‚Äì Device screen ki info deta hai
 console.log(screen.width); // Screen ki width
 console.log(screen.height); // Screen ki height

 4. location object ‚Äì Current URL se related operations karta hai
 console.log(location.href); // Current URL
 location.reload(); // Page reload karega

 5. history object ‚Äì Browser history ko access karta hai.
 history.back();  // ek page piche
 history.forward(); // ek page aage

```
